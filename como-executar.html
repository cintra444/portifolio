Para acessar essa aplicação, será necessário executar o arquivo .jar.

Como executar o arquivo .jar

  1. Baixe o Java se ainda não tiver instalado: java.com

  2. Baixe o arquivo projeto_final.jar

  3. Abra o terminal (windows+r e digite "cmd")

  4. Execute o comando: 
    java -jar projeto_final.jar

Se o Java estiver instalado corretamente, o arquivo .jar será executado.

Se você encontrar algum erro, verifique se o Java está instalado e configurado corretamente no seu sistema.
